# Описание системы для коммерческого предложения

## Общая концепция

Система состоит из двух основных частей:

1. **Сервис сбора цен** — автоматически собирает актуальные цены на товары с разных маркетплейсов
2. **Telegram-приложение** — удобный интерфейс для пользователей, где они могут найти лучшие цены на товары

⸻

## Часть 1: Сервис сбора цен

### Что делает сервис

Сервис работает в фоновом режиме и автоматически:

- **Собирает информацию о товарах** с указанных маркетплейсов
- **Обновляет цены каждый час**, чтобы данные всегда были актуальными
- **Хранит всю информацию** в удобном виде для быстрого доступа

### Что нужно предоставить

Для работы сервиса потребуется список товаров с указанием:
- Название товара
- Картинка товара
- Категория товара
- Ссылка на товар на каждом маркетплейсе

### Что получаете

- **Актуальные цены** на все товары с разных маркетплейсов
- **Автоматическое обновление** данных каждый час
- **Готовые данные** для отображения в Telegram-приложении

⸻

## Часть 2: Telegram-приложение

### Путь пользователя

#### 1. Первое открытие (онбординг)

**Экран 1: Приветствие**
- Краткая анимация «Время чистых эмоций»
- Автоматический переход на следующий экран

**Экран 2: Знакомство**
- Объяснение, что приложение помогает найти лучшие цены
- Кнопка «Далее»

**Экран 3: Регистрация**
- Форма для ввода данных:
  - ФИО
  - Телефон
  - Email
  - Согласие на обработку данных
- Кнопки «Назад» и «Подтвердить»

**Экран 4: Демонстрация**
- Автоматический поиск лучшей цены на пример товара
- Анимация процесса поиска

**Экран 5: Результаты демонстрации**
- Показ результатов поиска с разных маркетплейсов
- Возможность посмотреть детали товара

**Экран 6: Карточка товара**
- Детальная информация о товаре
- Кнопка для перехода на маркетплейс

После завершения онбординга пользователь попадает на главный экран.

#### 2. Основной функционал

**Главный экран**
- Простой интерфейс с одной большой кнопкой «Поиск»
- Возможность показа акций / новостей (опционально)

**Поиск товаров**

1. **Выбор категории**
   - Список доступных категорий товаров
   - Например: «Таблетки для посудомоечных средств», «Средства для стирки»

2. **Выбор товара**
   - Список товаров в выбранной категории
   - Карточки с названием и изображением товара

3. **Поиск цен**
   - Анимация процесса поиска
   - Автоматический поиск цен на всех маркетплейсах

4. **Результаты поиска**
   - Список всех маркетплейсов с ценами на выбранный товар
   - Сортировка от самой низкой цены к высокой
   - Возможность посмотреть детали по каждому маркетплейсу

5. **Карточка товара**
   - Полная информация о товаре: название, изображение, цена
   - Название маркетплейса
   - Кнопка «Купить» для перехода на маркетплейс

### Навигация

- **Простая навигация** между экранами
- **Кнопка «Назад»** на каждом экране для возврата
- **Автоматический возврат** на главный экран после перехода на МП или закрытия карточки товара

### Обработка ситуаций

- **Ошибки сети** — показ понятного сообщения с возможностью повторить
- **Нет результатов** — сообщение о том, что предложений нет

⸻

## Компоненты системы

### 1. База данных для цен

**Хранение информации:**
- Список всех товаров
- Категории товаров
- Цены с каждого маркетплейса

**Преимущества:**
- Быстрый доступ к данным
- Надёжное хранение информации

### 2. Система сбора цен

**Функции:**
- Автоматический сбор данных с маркетплейсов
- Обработка различных форматов страниц
- Обход защиты от автоматического сбора
- Автоматический запуск каждый час
- Логирование всех операций для контроля работы

**Результат:**
- Актуальные данные всегда готовы для использования
- Автоматическая работа без участия человека

### 3. База данных пользователей

**Хранение информации:**
- Данные пользователей (ФИО, телефон, email)
- Статус прохождения онбординга
- История поисков (опционально)
- Согласия на обработку данных

**Безопасность:**
- Защита персональных данных
- Соответствие требованиям по обработке данных

### 4. Серверная часть приложения

**Функции:**
- Обработка запросов от пользователей
- Предоставление данных о товарах и ценах
- Управление пользователями
- Интеграция с Telegram
- Обеспечение безопасности
- Интеграция с Mindbox для отправки данных пользователей (ФИО, телефон, email) в систему лояльности

### 5. Интерфейс приложения

**Экраны:**
- Онбординг (6 экранов)
- Главный экран
- Поиск товаров (категории, товары, результаты, карточка)

**Особенности:**
- Удобная навигация
- Красивые анимации
- Адаптивный дизайн
