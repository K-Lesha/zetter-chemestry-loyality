Описание сервиса парсинга цен

Цель сервиса

Сервис собирает актуальную информацию о товарах с разных маркетплейсов и хранит её в базе данных. Эта информация используется вашим Telegram-приложением для отображения цен и ссылок на товары.

⸻

Что делает сервис
	1.	Получает список всех товаров с ссылками, разбитых по категориям.
	2.	Заходит на маркетплейсы и парсит данные по каждому товару:
	•	Название товара
	•	Картинку товара
	•	Категорию товара
	•	Ссылку на маркетплейс
	•	Цену
	3.	Сохраняет всю собранную информацию в базу данных.
	4.	Обновляет данные раз в час, чтобы информация была всегда актуальной.

⸻

Что нужно от клиента (то есть что должен предоставить пользователь сервиса)
	•	Список товаров для отслеживания с указанием:
	•	Название товара
	•	Картинка (URL или файл)
	•	Категория, к которой относится товар
	•	Ссылка на маркетплейс, где продается товар

Это минимальный набор данных, чтобы парсер знал, что искать и где.

⸻

Что делает разработчик
	1.	Настраивает парсер, который:
	•	Проходит по всем ссылкам, предоставленным клиентом
	•	Собирает актуальные данные о товаре (цена, наличие, название, картинка)
	2.	Добавляет результаты парсинга в таблицу/базу данных, чтобы Telegram-приложение могло считывать актуальные цены и ссылки.
	3.	Настраивает автоматический запуск парсера каждый час, чтобы данные обновлялись без ручного вмешательства.

⸻

Результат работы
	•	Telegram-приложение получает актуальные данные по товарам: название, картинку, цену и ссылку на маркетплейс, разделенные по категориям.
	•	Пользователь всегда видит самые свежие цены, а разработчик имеет готовую, регулярно обновляемую базу данных для работы бота.
